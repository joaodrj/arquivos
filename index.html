Instalação do MicroPython na AmadoBoard

Esta página permite instalar o MicroPython em sua placa AmadoBoard de maneira fácil e rápida. Basta conectar a sua placa via USB e clicar no botão abaixo para iniciar a instalação.

Seu navegador não suporta a Web Serial API. Utilize o Google Chrome ou o Microsoft Edge.

Se você não encontrar seu dispositivo na lista, talvez precise instalar os drivers apropriados: CP2102 Driver ou CH341 Driver.

Baseado em ESP Web Tools – GitHub

Desenvolvido para facilitar a instalação do MicroPython na sua placa ESP32.
